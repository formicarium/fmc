# Usage
<!-- usage -->
```sh-session
$ npm install -g @formicarium/cli
$ fmc COMMAND
running command...
$ fmc (-v|--version|version)
@formicarium/cli/1.4.13 darwin-x64 node-v10.8.0
$ fmc --help [COMMAND]
USAGE
  $ fmc COMMAND
...
```

## `fmc help [COMMAND]`

display help for fmc

```
USAGE
  $ fmc help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/v2.1.1/src/commands/help.ts)_
